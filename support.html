Need help?

Community resources for answering your AUI React-Bootstrap questions.

Stay up to date on the development of React-Bootstrap and reach out to the community with these helpful resources.

Stack Overflow

Ask questions about specific problems you have faced, including details about what exactly you are trying to do. Make sure you tag your question with aui-react-bootstrap. You can also read through existing AUI React-Bootstrap questions.

As this project is heavily based on React Bootstrap make sure you also check the questions with the react-bootstrap tag and read through existing React-Bootstrap questions.

Chat rooms

Discuss questions in the #aui-react-bootstrap channel on the Reactiflux Slack or on Gitter.

GitHub issues

The issue tracker is the preferred channel for bug reports, features requests and submitting pull requests. See more about how we use issues in the contribution guidelines.
